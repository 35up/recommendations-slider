## Customization

<br/>

### With Recommendations Height

<Story id="recommendations-slider--with-recommendations-height" />

```jsx
  <tfup-recommendations-slider
    base-product={JSON.stringify({
      title: 'Apple iPhone 12 Pro - Pacific blue',
    })}
    seller="35up-test"
    style="--recommendation-height:20rem;"
  ></tfup-recommendations-slider>
```

### With Font Size

<Story id="recommendations-slider--with-font-size" />

```jsx
  <tfup-recommendations-slider
    base-product={JSON.stringify({
      title: 'Apple iPhone 12 Pro - Pacific blue',
    })}

```
